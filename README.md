
# Instalaciones

1. Instalá Node JS

### Frontend

mkdir login-react-mysql-node
cd login-react-mysql-node
npm create vite@latest
name: frontend
cd frontend
npm install
npm run dev
npm install axios

### Backend

cd login-react-mysql-node
mkdir backend
cd backend
npm init -y
npm install express
npm install mysql
npm install cors

### Para Ejecutar

- Para el Frontend tener en ejecución con `npm run dev`
- Para el Backend ejecutar `node server.js`
